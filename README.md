# 📊 Student Performance Analysis

## 📌 Overview

This project was completed as **Task 1** of my **Data Science & Data Analysis with Python Internship**. The objective was to perform Exploratory Data Analysis (EDA) on the Student Performance dataset, extract meaningful insights, and visualize the results using Python.

The project demonstrates the complete data analysis workflow, including data loading, exploration, cleaning, statistical analysis, and visualization.

---

## 🎯 Objectives

- Load and explore the Student Performance dataset.
- Perform basic data cleaning and validation.
- Analyze student academic performance.
- Identify factors affecting final grades.
- Create visualizations for better understanding.
- Document findings with Markdown explanations.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## 📂 Dataset

**Dataset:** Student Performance Dataset (Mathematics)

The dataset contains information about students' demographic details, study habits, family background, and academic performance.

**Key Features:**

- Gender
- Age
- Study Time
- Travel Time
- Previous Grades (G1, G2)
- Final Grade (G3)
- Number of Failures
- Absences
- Family Relationship
- Health

---

## 📈 Project Workflow

1. Import required Python libraries.
2. Load the dataset.
3. Explore the dataset.
4. Check data types and missing values.
5. Calculate average final grades.
6. Count students scoring above 15.
7. Perform correlation analysis.
8. Compare student performance by gender.
9. Create visualizations.
10. Summarize insights.

---

## 📊 Visualizations

The notebook includes:

- 📌 Bar Chart – Average Final Grade by Gender
- 📌 Histogram – Distribution of Final Grades
- 📌 Scatter Plot – Study Time vs Final Grade
- 📌 Bar Chart – Travel Time vs Average Final Grade

---

## 🔍 Key Insights

- Average Final Grade: **10.42**
- **40 students** scored above 15 marks.
- Previous grades (**G1** and **G2**) have the strongest positive correlation with the final grade.
- Students with more previous failures generally achieve lower final grades.
- Male students have a slightly higher average final grade than female students in this dataset.
- Study time shows a weak positive relationship with academic performance.
