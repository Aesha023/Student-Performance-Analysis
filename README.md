# Student-Performance-Analysis
Data Science Internship Task 1 - Student Performance Analysis using Python, Pandas and Matplotlib.

## Overview
This project analyzes the Student Performance dataset using Python.

## Objectives
- Explore the dataset
- Clean the data
- Perform statistical analysis
- Visualize insights
- Answer internship questions

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Dataset
Student Performance Dataset (UCI Machine Learning Repository)

## Results
- Average Final Grade: 10.42
- Students scoring above 15: 40
- Previous grades (G1, G2) showed the strongest positive correlation with final grades.
- Male students had a slightly higher average final grade in this dataset.
